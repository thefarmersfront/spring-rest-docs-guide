== 주문(Order)
주문을 등록하고 `결제(PAID) -> 출하(SHIPPED) -> 배송완료(COMPLETED)` 처리를 할 수 있습니다.

=== 주문검색
주문정보를 회원번호(memberNo)과 주문번호(orderNo)를 이용해서 검색할 수 있습니다.
페이징 처리된 목록을 제공합니다.

|====
|속성 |설명

|`memberNo` |회원번호
|`orderNo` |주문번호
|`page` |페이지(0 부터 시작)
|`size` |페이지당 조회건수(최소: 1, 최대: 1000)

|====
